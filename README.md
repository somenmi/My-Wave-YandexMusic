**My Wave** – расширение для браузера, которое добавляет регулируемые фильтры для анимаций и вырубает объекты в разделе «Моя волна» Яндекс Музыки.

- 🎨 Наложение цветовых фильтров на анимационные элементы
- 🎛️ Регулировка: Ч/Б; Яркость; Сепия; HUE; Насыщенность;
- 🖌️ Смена цветовой гаммы объектов в реальном времени
- 💾 Сохранение настроек между сессиями

---

## 🛠 Установка

1. Скачайте файлы
2. В chrome://extensions включите "Режим разработчика"
3. Загрузите распакованное расширение
4. Откройте расширение навкладке Яндекс Музыки
5. Наслаждайтесь

## 💾 Хранение данных

Настройки фильтров сохраняются в `localStorage` домена Яндекс Музыки. При повторном открытии страницы все параметры восстанавливаются автоматически.

## 🔧 Системные требования

- Браузер на базе Chromium (Chrome, Edge, Opera)
- 256 МБ RAM
- 10 МБ свободного места
